There's always lots to learn about the Dev Tools in your browser. Most browsers (despite their branding differences) use the same underlying engines. They fall into several families:

| Common Family Name | Rendering Engine | Major Browsers |
|--------------------|--------|----------------|
| <a href="https://www.chromium.org/Home/" target="_blank">Chromium-Based</a> | <a href="https://www.chromium.org/blink/" target="_blank">Blink</a>  | Google Chrome, Microsoft Edge, Android WebView, Opera, Brave, etc. |
| Firefox Family     | <a href="https://firefox-source-docs.mozilla.org/overview/gecko.html" target="_blank">Gecko</a>  | Firefox, Tor, LibreWolf, etc.    |
| Apple Ecosystem    | <a href="https://developer.apple.com/documentation/webkit" target="_blank">WebKit</a> | Safari, GNOME Web                |
