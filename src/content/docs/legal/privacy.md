*Last updated: April 12, 2026*

## Overview

This documentation site is intended as a public reference resource. Protecting visitor privacy is important, and the site is designed to minimize the collection of personal information.

## Personal Information

This site does not collect personally identifiable information from visitors.

Visitors are not required to create accounts, submit personal details, or provide identifying information in order to read the documentation.

## Analytics and Tracking

At this time, this site does not use analytics services or visitor-tracking tools.

If analytics or similar tools are introduced in the future, this policy will be updated accordingly.

## Cookies

This site does not intentionally set cookies for tracking or identification purposes.

However, basic technical cookies or similar mechanisms may be used by the hosting provider or content delivery infrastructure to ensure reliable operation of the website.

## Hosting Provider

This site is hosted using GitHub Pages.

As part of normal operation, GitHub may automatically collect limited technical information such as IP addresses, browser details, request timestamps, and requested resources for operational, security, and reliability purposes. This information is handled according to GitHub’s own privacy practices.

The site owner does not control or access this infrastructure-level data.

## Server Logs

Like most websites, the hosting environment may automatically record limited technical information such as:

* IP address
* browser type
* requested pages
* date and time of access

This information is used for operational and security purposes and is not used by the site owner to identify individual visitors.

## External Links

This site may include links to third-party websites for reference or convenience.

External sites operate independently and may collect information according to their own privacy policies. Visitors should review the privacy policies of any third-party sites they access.

## Embedded Content

If pages on this site include embedded material from third-party providers (such as videos, diagrams, or code hosting platforms), those providers may collect technical usage data as if the visitor accessed their site directly.

## Future Changes

This site is currently generated as a static documentation site. If site features change in the future (for example, analytics, interactive services, or user accounts), this Privacy Policy will be updated to reflect those changes.

## Contact

If you have questions about this Privacy Policy, please contact the site owner.
