### VS Code

Here's a snippet I created to make adding a `<link>` for PicoCSS easy.

```json
{
    "PicoCSS Link": {
        "prefix": "pico",
        "body": [
                "<link rel='stylesheet' href='https://cdn.jsdelivr.net/npm/@picocss/pico@2/css/pico.min.css'>$0"
        ],
        "description": "Stylesheet link for PicoCSS via CDN"
    }
}
```
