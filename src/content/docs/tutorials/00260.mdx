- Create a web page
- Discover the browser's Dev Tools
- Introduce you to the DOM through the `document` object

```js title="Type into the Dev Tool's Console"
document.head
document.head.title = 'A Living Document'
document.body

// Darker, but not black background - just enough that we can still make out the text
document.body.style.backgroundColr = '#'
// Nice complementary color
document.body.style.color = '#'
```


The browser uses the HTML and CSS like a "recipe", one that tells the browser what content is on the site and how it should appear to the user.

```html title="index.html"
<!DOCTYPE html>
<html lang="en">
<head>
    <title>HTML, CSS and JS</title> 
    <style>
      h1 {
        color: rebeccapurple;
      }
    </style>
    <script>
      console.log('Hello Browser Dev Tools!');
    </script>
</head>
<body>
    <h1>Hello Browser</h1>    
</body>
</html>
```

```html title="index.html"
<!DOCTYPE html>
<html lang="en">
<head>
    <title>HTML, CSS and JS</title> 
    <link rel="stylesheet" href="css/styles.css">
    <script src="js/main.js"></script>
</head>
<body>
    <h1>Hello Browser</h1>    
</body>
</html>
```

```js title="js/main.js"

```

```css title="css/styles.css"

  color: rebeccapurple;
}
```
