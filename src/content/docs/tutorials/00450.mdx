In this tutorial, we'll bring together all the common pieces of a website: HTML, CSS, Images, and (of course) ***JavaScript***. The goal of this tutorial is for you to simply see how the pieces could fit together to become a unified whole. We'll start from scratch and build this in four phases.

## Project Setup

The first phase is planning what files we'll need and where we'll put them. Our folder structure will look like this.

<FileTree>
  - AllTogether Root of your website
    - css/ 
      - styles.css
    - img/
      - construction.webp
      - internet.webp
      - planning.webp
    - js/
      - main.js
    - **index.html** The "home" page of the site
</FileTree>

Follow these quick steps to create the text files for the site. We'll just keep them as empty files at this point. We'll put actual content in them at later phases of this tutorial.

<Steps>
1. Create a folder called `AllTogether` as the place for our website files.
1. Create an `index.html` at the root of that folder.
1. Create a `styles.css` under a folder called `css`.
1. Create a `main.js` under a folder called `js`.
1. Add a couple of images by saving these under a folder called `img`.
</Steps>

The browser uses the HTML and CSS like a "recipe", one that tells the browser what content is on the site and how it should appear to the user.

```html title="index.html"
<!DOCTYPE html>
<html lang="en">
<head>
    <title>HTML, CSS and JS</title> 
    <style>
      h1 {
        color: rebeccapurple;
      }
    </style>
    <script>
      console.log('Hello Browser Dev Tools!');
    </script>
</head>
<body>
    <h1>Hello Browser</h1>    
</body>
</html>
```

```html title="index.html"
<!DOCTYPE html>
<html lang="en">
<head>
    <title>HTML, CSS and JS</title> 
    <link rel="stylesheet" href="css/styles.css">
    <script src="js/main.js"></script>
</head>
<body>
    <h1>Hello Browser</h1>    
</body>
</html>
```

```js title="js/main.js"
console.log('Hello Browser Dev Tools!');
```

```css title="css/styles.css"

  color: rebeccapurple;
}
```
