- [ ] Bingo basics

Bingo Cards: Each player receives a Bingo card, typically a 5x5 grid with the letters B-I-N-G-O across the top. Each column corresponds to a specific range of numbers:

- **B**: 1-15
- **I**: 16-30
- **N**: 31-45 (with a free space in the center)
- **G**: 46-60
- **O**: 61-75


## Generating Bingo Cards

The Bingo card will be an HTML table with a `<thead>` for the "BINGO" letters and a `<tbody>` for the rows of numbers.

```js title="css/styles.css"
@import "tailwindcss";

/* Point Tailwind at where your HTML lives */
@source "../**/*.html";

@layer components {
  /* Root BINGO table */
  table[data-bingo-card] {
    @apply border-collapse mx-auto text-center text-sm;
  }

  /* Header: B I N G O letters */
  table[data-bingo-card] thead {
    th {
      @apply px-4 py-1 border border-slate-500 font-bold tracking-wide 
             bg-slate-100 text-base;
      /* cells stay as wide as their content by default; no width applied */
      @apply whitespace-nowrap;
    }
  }

  /* Body rows: numbers */
  table[data-bingo-card] tbody {
    td {
      @apply px-2 py-1 border border-slate-400 align-middle 
             whitespace-nowrap select-none;
    }
    td.free {
        @apply px-0 mx-auto font-stretch-condensed text-sm font-bold;
    }
    /* Optional: subtle hover so players can “mark” a cell visually */
    td:hover {
      @apply bg-yellow-100 cursor-pointer italic;
    }
    td.free:hover {
      @apply bg-orange-100 cursor-pointer;
    }

    /* Optional: slightly larger text for readability */
    td[data-value] {
      @apply font-semibold text-base;
    }
  }
}
```
